# # Student Management System (JDBC)

A simple console-based Student Management System built using Java, JDBC, and MySQL.

## Features

* Add Student
* View All Students
* Update Student Marks
* Delete Student Records
* Menu-Driven Interface
* Database Connectivity using JDBC
* PreparedStatement for secure SQL operations

## Technologies Used

* Java
* JDBC
* MySQL
* MySQL Workbench
* VS Code

## Database Structure

Table: `student`

| Column | Type                              |
| ------ | --------------------------------- |
| id     | INT (AUTO_INCREMENT, PRIMARY KEY) |
| names  | VARCHAR                           |
| age    | INT                               |
| marks  | DOUBLE                            |

## Concepts Practiced

* JDBC Connection
* DriverManager
* PreparedStatement
* ResultSet
* CRUD Operations
* Exception Handling
* Scanner Input Handling

## Learning Outcome

This project helped me understand how Java applications interact with MySQL databases using JDBC and how to perform Create, Read, Update, and Delete (CRUD) operations in a real project.
